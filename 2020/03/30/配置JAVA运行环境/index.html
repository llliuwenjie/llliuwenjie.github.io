JAVA运行环境

发表于 2020-03-30 更新于 2021-01-15

MAC

java -version

which java

输出结果: /usr/bin/java
###显示java指令文件的属性

输出结果如下 lrwxr-xr-x 1 root wheel 74 3 17 14:43 /usr/bin/java -> /System/Library/Frameworks/JavaVM.framework/Versions/Current/Commands/java

从输出结果可以知道/usr/bin/java文件是一个链接文件，实际是指向/System/Library/Frameworks/JavaVM.framework/Versions/Current/Commands/java文件的

进入实际指令所在的文件夹

但是这个目录并不是JAVA_HOME目录

查看JDK javahome
